Anomalous egress traffic detected — M365 account, possible data exfiltration [CS-2024-88341]
* Description

CrowdStrike Falcon (Alert CS-2024-88341) triggered a high-severity anomalous egress detection at 14:27 today against an authenticated Microsoft 365 session for the account m.whitfield@cushwake.com.

The session transferred approximately 4.3 GB of data to an unresolved external IP address (185.220.101.47) over an 18-minute window via HTTPS port 443. This volume is approximately 13× the account's 30-day average daily egress. No MFA failure or brute-force attempt was detected — the session token at time of transfer was valid.

This account was flagged earlier today in INC0047799 for anomalous Active Directory reconnaissance activity (1,847 LDAP queries, 11:30 UTC). A JD Edwards financial system session was also recorded from the same workstation (CW-ATL-WKS073) at 13:55 UTC, approximately 30 minutes prior to the egress event.

Ticket escalated to James Huntley per IR SOP. No containment action taken pending direction. DFIR team (CrowdStrike Services) has been notified and is engaged under active retainer.
